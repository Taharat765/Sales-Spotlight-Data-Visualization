# Sales Insight View (Interactive Dashboard using Power BI)

## Project Objective
The goal of this project is to analyze overall sales performance and customer behavior to help the store owner make data-driven decisions and plan future sales strategies.

---

## Dataset Used
- Dataset containing order details, customer demographics, product information, and sales channels.

---

## Key Questions (KPIs)
- Compare sales and orders using a single chart.  
- Identify which month achieved the highest performance.  
- Analyze purchasing trends by gender.  
- Examine different order statuses.  
- Identify top contributing locations.  
- Explore the relationship between age and gender.  
- Find the channel contributing most to sales.  
- Determine the highest selling category.  
- Calculate the percentage of total orders delivered.

---

## Process
1. Cleaned and verified data for missing values or inconsistencies.  
2. Ensured proper data types and standardized formats.  
3. Built pivot tables for each analytical question.  
4. Designed an interactive Power BI dashboard integrating all insights with dynamic slicers.

---

## Dashboard Overview
- Visualizes revenue, order quantity, and customer distribution.  
- Provides trend analysis and top-performing products and customers.  
- Enables interactive exploration through slicers and filters.

---

## Project Insights
- Female customers showed higher purchasing activity.  
- Adult customers formed the majority of the buyer group.  
- Online sales channels performed the best overall.  
- Most orders were successfully delivered.

---

## Conclusion
The insights from this dashboard can guide the business toward improving its marketing strategies, focusing on the right customer segments, and optimizing sales channels.
